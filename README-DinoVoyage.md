# DinoVoyage — Dinosaur Science App

DinoVoyage is a Next.js-based educational app that explores dinosaurs and paleontology. This repository naming follows a brand-friendly, scalable pattern suitable for expansion to other science topics in the future.

## Why this name
- DinoVoyage conveys a journey through dinosaur knowledge, with room to expand into broader science education.
- The name is short, easy to pronounce, and suitable for branding, domain names, and social handles.

## Repository naming guidance
- Suggested GitHub repo slug: dino-voyage (lowercase, hyphenated)
- Alternatives if needed: dinovoyage, dino voyage (space not allowed in slug)

## Local development quick start
- Prerequisites: Node.js, npm/yarn/pnpm (as per your preference)
- Install dependencies: `npm ci` or `yarn install` or `pnpm install`
- Run dev server: `npm run dev` (or `yarn dev`, `pnpm dev`)
- Open http://localhost:3000

## Uploading to GitHub (high level)
- Create a new repository on GitHub with the name: dino-voyage
- In your local project, set the remote and push:
  - `git remote add origin https://github.com/<your-username>/dino-voyage.git`
  - `git branch -M main`
  - `git push -u origin main`

## CI/CD and quality setup (optional, quick starter)
- Add GitHub Actions workflow for Next.js build:
  - See .github/workflows/ci.yml (starter template in this repo)
- Consider adding a MIT license to maximize collaboration.
